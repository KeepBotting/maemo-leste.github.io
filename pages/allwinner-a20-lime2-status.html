Allwinner A20 LIME2 Status

Images

Pre-alpha LIME2 images are available here (named sunxi):

http://maedevu.maemo.org/images/

Note that they are not particularly usable, as we did not yet enable Mali GPU acceleration, and without touchscreen a mouse is not very usable, as the pointer is not (yet) visible.

Installation

This image is flashed to a microSD card in two parts: first the actual image with the kernel and root filesystem, followed by the u-boot bootloader.

u-boot should be flashed using the following:

# dd if=A20-OLinuXino-Lime2_defconfig.bin of=/dev/yourmicrosd bs=1024 seek=8
